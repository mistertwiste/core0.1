# GPT-4o Chat MVP

Ein simples Flask-MVP, das:
- Nutzereingaben per Textfeld entgegennimmt
- Ein “Prompt-Generator” verbessert die Eingabe
- Mit GPT-4o chatten (OpenAI API)
- Antwort im Browser ausgeben
